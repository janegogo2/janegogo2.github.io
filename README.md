# janegogo2.github.io
เป็นแค่การรวบรวมทีวีออนไลน์ โดยมีจุดประสงค์เพื่อประชาสัมพันธ์และส่งเสริมช่องรายการทีวีต่างๆ บนอินเตอร์เน็ตสำหรับผู้ที่ไม่สะดวกในการรับชมผ่านเครื่องรับโทรทัศน์เท่านั้น ไม่มีการทำซ้ำหรือเปลี่ยนแปลงใดๆที่ส่งผลกับช่องทีวีนั้นๆ การกระทำนี้มิได้มีวัตถุประสงค์เป็นการแสวงหาประโยชน์หรือสนับสนุนการกระทำผิดต่อกฎหมายใดทั้งสิ้น
